# healthcare_predictive_analysis
Analyse exploratoire + ML dataset healthcare | EDA + XGBoost 85% accuracy. Data from kaggle
**Médecin Data Scientist** | EDA + Machine Learning sur dataset hôpital Goma (10k+ admissions)

## Objectif Projet
Analyse complète données healthcare simulées **Hôpital Goma 2025** :
- **Nettoyage** 10k+ admissions (duplicatas, NaN)
- **EDA** : Top diagnostics, coûts, durée séjour
- **ML** : Prédiction urgences (RandomForest 85% accuracy)
